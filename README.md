### Method 1: Direct APK Download

2. Download the latest `FunMessageApp.apk` file

1. Click on `FunMessageApp.apk` in the repository

## 🔧 Development Setup

If you want to build the app from source:

### Prerequisites
- Flutter SDK (3.19.6 or later)
- Android Studio with Android SDK
- Git

### Building from Source
```bash
# Clone the repository
git clone https://github.com/LengendGamerYT/Message-Fun-App.git
cd Message-Fun-App

# Navigate to the Flutter project
cd minimal_message_app

# Get dependencies
flutter pub get

# Build APK
flutter build apk --release
```
